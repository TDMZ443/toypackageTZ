# Scales a numeric vector using min_max scaling method

Scales a numeric vector using min_max scaling method

## Usage

``` r
min_max_scale(x)
```

## Arguments

- x:

  A numeric vector

## Value

A scaled numeric vector

## Examples

``` r
x <- 1:5
min_max_scale(x)
#> [1] 0.00 0.25 0.50 0.75 1.00
```
